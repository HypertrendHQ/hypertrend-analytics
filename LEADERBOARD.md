## 🔧 核心工具

### get_leaderboard(type, period, limit)
获取各类榜单

**参数:**
- `type`: 榜单类型
  - `gravity` - 引力指数榜
  - `pnl` - 收益榜
  - `winrate` - 胜率榜
  - `copytrading` - 带单榜
  - `gainers` - 涨幅榜
  - `rising` - 新贵榜
- `period`: 时间周期 (`1d`, `7d`, `30d`, `all`)
- `limit`: 返回数量 (默认10, 最大100)

**示例:**
```javascript
get_leaderboard("gravity", "all", 10)
get_leaderboard("pnl", "7d", 20)
get_leaderboard("winrate", "30d", 50)
```

### get_leaderboard_changes(type, period)
获取榜单变化

**参数:**
- `type`: 榜单类型
- `period`: 变化周期 (`1d`, `7d`)

**返回:**
- 排名变化列表
- 新晋/跌出榜单
- 变化原因分析

**示例:**
```javascript
get_leaderboard_changes("gravity", "1d")
```

### get_address_ranking(address)
查询特定地址的排名情况

**示例:**
```javascript
get_address_ranking("0x7a3f...9d2e")
```

**返回:**
```json
{
  "address": "0x7a3f...9d2e",
  "gravity_rank": 1,
  "gravity_score": 982,
  "pnl_7d_rank": 1,
  "pnl_7d": "+$125,680",
  "winrate_rank": 5,
  "winrate": "87.6%",
  "copytrading_rank": 1,
  "overall_rank": 1
}
```

### compare_rankings(addresses, metric)
对比多个地址的排名

**示例:**
```javascript
compare_rankings(["0x7a3f", "0x9c1b", "0x3d2e"], "gravity")
```

### subscribe_leaderboard_updates(type, callback)
订阅榜单实时更新

**示例:**
```javascript
subscribe_leaderboard_updates("gravity", (update) => {
  console.log("引力指数榜更新:", update)
})
```

## 📋 API 接口

### 获取榜单
```
GET /v1/leaderboard/{type}
Parameters:
  - period: 1d, 7d, 30d, all
  - limit: 1-100
  - min_gravity: 最小引力指数

Response:
{
  "type": "gravity",
  "period": "all",
  "updated_at": "2026-03-11T14:30:00Z",
  "data": [
    {
      "rank": 1,
      "address": "0x7a3f...9d2e",
      "gravity_index": 982,
      "hexagram": {...},
      "pnl_7d": "+125680",
      "winrate": "87.6%",
      "rank_change": 4,
      "is_new": false
    }
  ]
}
```

### 获取榜单变化
```
GET /v1/leaderboard/{type}/changes
Parameters:
  - period: 1d, 7d

Response:
{
  "rising": [...],      // 排名上升
  "falling": [...],     // 排名下降
  "new_entries": [...], // 新晋
  "dropped": [...],     // 跌出
  "analysis": "..."     // 变化分析
}
```

- ⚠️ 新贵数据样本少，需谨慎

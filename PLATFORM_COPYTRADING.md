## 🛡️ 风控机制

### 平台级风控

HyperTrend 平台自动执行:

- ✅ 带单者实名认证
- ✅ 交易行为监控
- ✅ 异常交易预警
- ✅ 资金安全托管
- ✅ 保险基金保障

### 用户级风控

用户可自定义:

```toml
[hypertrend.risk]
# 仓位控制
max_position_per_leader = 0.4    # 单个带单者最多40%
max_total_position = 0.8         # 总仓位最多80%
min_cash_ratio = 0.2             # 最少保留20%现金

# 止损止盈
stop_loss = 0.15                 # 15%止损
take_profit = 0.30               # 30%止盈
trailing_stop = 0.10             # 10%跟踪止损

# 风险监控
max_daily_loss = 0.05            # 日亏损不超过5%
max_drawdown = 0.25              # 最大回撤25%
auto_pause_on_volatility = true  # 高波动自动暂停
```

### 带单者风控指标

每个带单者显示风险指标:

| 指标 | 说明 | 预警值 |
|------|------|--------|
| 最大回撤 | 历史最大亏损 | >20% 警告 |
| 夏普比率 | 风险调整收益 | <1.0 警告 |
| 胜率 | 盈利交易占比 | <50% 警告 |
| 杠杆使用率 | 平均杠杆倍数 | >15x 警告 |
| 爆仓次数 | 历史爆仓记录 | >0 警告 |

Response: {
  "account_id": "HT_7a3f9d2e",

  "signature_expiry": "2026-03-18T14:30:00Z",
  "token": "eyJhbGciOiJIUzI1NiIs..."

### 刷新签名
```
POST /v1/account/refresh
Headers: {
  "Authorization": "Bearer {token}"
}
Body: {
  "new_signature": "0x9b3c...8e2f",
  "new_timestamp": "2026-03-18T14:30:00Z"
}
```

### 获取带单者列表
```
GET /v1/leaders?min_gravity=800&period=30d&limit=20
Headers: {
  "Authorization": "Bearer {token}"
}
```

### 开始跟单
```
POST /v1/copytrading/start
Headers: {
  "Authorization": "Bearer {token}"
}
Body: {
  "leader_id": "0x7a3f...9d2e",
  "amount": 5000,
  "options": {
    "stop_loss": 0.15,
    "take_profit": 0.30
  }
}
```

### 获取跟单状态
```
GET /v1/copytrading/status
Headers: {
  "Authorization": "Bearer {token}"
}
```

### 停止跟单
```
POST /v1/copytrading/stop
Headers: {
  "Authorization": "Bearer {token}"
}
Body: {
  "leader_id": "0x7a3f...9d2e",
  "close_positions": true
}
```

**认证说明:**
- 所有 API 请求需要在 Header 中携带 `Authorization: Bearer {token}`
- Token 在连接账户时返回，有效期与签名一致 (7天)
- 签名过期后需要重新调用 `/v1/account/connect` 获取新 Token

## 🔧 核心工具

### add_address_monitor(address, options)
添加地址监控

```javascript
add_address_monitor("0x7a3f...9d2e", {
  monitor_types: ["position", "trade", "balance", "pnl", "risk"],
  min_notify_amount: 100,
  leverage_threshold: 2,
  pnl_threshold: 500,
  risk_threshold: 10,
  check_interval: 30,
  notify_channels: ["telegram", "chat"]
})
```

### get_monitor_list()
获取监控列表

```javascript
get_monitor_list()
```

### get_address_monitor_detail(address, period)
获取地址监控详情

```javascript
get_address_monitor_detail("0x7a3f...9d2e", "7d")
```

### pause_address_monitor(address)
暂停监控

```javascript
pause_address_monitor("0x7a3f...9d2e")
```

### resume_address_monitor(address)
恢复监控

```javascript
resume_address_monitor("0x7a3f...9d2e")
```

### remove_address_monitor(address)
移除监控

```javascript
remove_address_monitor("0x7a3f...9d2e")
```

### update_monitor_settings(address, options)
更新监控设置

```javascript
update_monitor_settings("0x7a3f...9d2e", {
  min_notify_amount: 200,
  pnl_threshold: 1000
})
```

### get_monitor_alerts(address, period)
获取预警历史

```javascript
get_monitor_alerts("0x7a3f...9d2e", "24h")
```

### compare_monitored_addresses(addresses, metric)
对比多个监控地址

```javascript
compare_monitored_addresses(
  ["0x7a3f", "0x9c1b", "0x2d8f"],
  "pnl_7d"
)
```

### export_monitor_data(address, format, period)
导出监控数据

```javascript
export_monitor_data("0x7a3f...9d2e", "csv", "30d")
```
